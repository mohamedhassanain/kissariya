# KissariyaMaroc

## À propos du projet

KissariyaMaroc est la plateforme n°1 au Maroc pour digitaliser votre commerce. Créez votre kissariya digitale et vendez sur WhatsApp en moins de 2 minutes.

## Technologies utilisées

Ce projet est construit avec :

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Backend & Auth)

## Installation locale

1. Clonez le dépôt
2. Installez les dépendances :
```sh
npm install
```
3. Lancez le serveur de développement :
```sh
npm run dev
```

## Déploiement

Le projet peut être déployé sur n'importe quelle plateforme de hosting statique (Vercel, Netlify, GitHub Pages) en connectant votre instance Supabase.
